## Cross-cutting lessons earned this session

(See also `log/incidents/2026-04-23_wave0-session-lessons.md` + memory files.)

1. **Coverage rules need coverage checks.** R7's presence grep missed every R10 violation. R7b's spot-check caught them all. Applies to any "every X must have Y" rule.
2. **Self-report is not evidence.** Agents' final messages said "all 21 criteria pass" while 6/6 files had critical issues. Always run an independent check.
3. **Sourcing fidelity fails systematically.** Hallucinated URLs, fabricated verbatim quotes, wrong page numbers, mis-labeled data series — not scattered errors but a pattern that requires prompt-level countermeasures (R11–R14).
4. **Wave-to-wave cadence matters.** Advisor-caught issues in Wave 0 saved Wave 1 from 4× multiplication. Red-team-caught issues in Wave 1 saved Waves 2 + 3. Each inspection is cheap vs the failure it prevents.
5. **Parallelism is a moving target.** User raised to 4, then dropped to 2. Always re-read the most recent instruction. Save to memory, not conversation.
6. **Worked examples need live arithmetic.** § 7 narrative plus § 8c chart can disagree numerically — both look plausible until the RC sum ≠ 100%. Require re-execution (R14).
7. **"Patch in place" is a judgment call bound to 0-CRITICAL.** The Wave 0 retro directive was conditional on 0-CRITICAL remaining true; it did for 1.1 (applied), didn't for 2.2 (re-spawned instead).
